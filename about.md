# 关于本站

本站点是基于 GitHub Pages 构建的静态网站，主要用于展示 IT 行业每日简报。

## 🎯 目的

- 测试 GitHub Pages 部署流程

## 🛠 技术栈

- **静态站点生成器**: Jekyll
- **托管平台**: GitHub Pages
- **主题**: Minima (GitHub Pages 默认主题)
