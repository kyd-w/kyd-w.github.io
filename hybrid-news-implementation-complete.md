## 🔧 技术实现

### 三层数据源架构

```
┌─────────────────────────────────────────┐
│ 第一层：RSS 源（70-80%）                 │
│  • IT之家、36氪、机器之心                │
│  • 天然时效性，权威质量                  │
└─────────────────────────────────────────┘
                    ↓ 不足时
┌─────────────────────────────────────────┐
│ 第二层：NewsAPI（10-20%）                │
│  • 补充国际新闻                          │
│  • 覆盖特定技术话题                      │
└─────────────────────────────────────────┘
                    ↓ 仍不足时
┌─────────────────────────────────────────┐
│ 第三层：搜索引擎（0-10%，备选）           │
│  • Tavily 搜索                           │
│  • 仅在绝对必要时使用                     │
└─────────────────────────────────────────┘
```

### 智能去重机制

1. **标题去重**: 标题相似度 > 85% 视为重复
2. **域名去重**: 同一域名的新闻只保留最新的
3. **时间过滤**: IT≤2天，AI≤3天
