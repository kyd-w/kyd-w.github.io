# Heartbeat Checklist

## 每日简报更新流程

### 1. 基础简报（每天）
- 时间/日期确认
- 今日天气（北京地区，默认）
- 今日日程/提醒

### 2. 行业动态（每周一更新）
**更新周期**：每周一生成完整版，仅周一推送到微信
**触发条件**：周一 或 中间有重大突发新闻
**时效要求**：仅保留2周内的信息，超过2周的旧闻删除

**内容模板**：
```markdown
## 🏢 数据中心行业动态

### 政策与趋势
• [标题]：[内容摘要]
  - 📎 [链接]

### 核心厂商动态
#### 算力厂商
• [厂商]：[内容]
  - 📎 [链接]

#### UPS厂商
• [厂商]：[内容]
  - 📎 [链接]

#### 制冷厂商
• [厂商]：[内容]
  - 📎 [链接]

#### 电池厂商
• [厂商]：[内容]
  - 📎 [链接]

### 技术前沿
• [内容]
  - 📎 [链接]
```

**搜索关键词**（周一执行）：
使用 web_search 时添加 date_after 参数过滤2周内信息：
- 数据中心 政策 PUE 工信部 + date_after=[14天前]
- 英伟达 华为 昇腾 数据中心 + date_after=[14天前]
- 维谛 华为 UPS 数据中心 + date_after=[14天前]
- 液冷 数据中心 浸没式 冷板 + date_after=[14天前]
- 钠电池 磷酸铁锂 数据中心 + date_after=[14天前]

**周二-周五逻辑**：
- 不推送行业动态（除非有重大突发新闻）
- 如有突发，临时推送并标注「🚨 突发」
- 每日简报仅包含基础信息（时间/天气/日程）

---

### 3. 重大突发新闻识别（手动触发）

**触发条件**：你主动发消息询问，例如：
> "帮我看下这个是不是重大新闻：[内容]"

**重大突发标准**：

#### ✅ 会触发推送（周一外临时推送）
- **政策类**：国家部委（发改委/工信部/能源局）发布的重大政策
  - 例：数据中心PUE标准调整、东数西算节点扩容、强制液冷时间表变化
- **厂商类**：头部厂商重大产品发布/中标（华为/维谛/英伟达/宁德时代）
  - 例：华为发布新一代昇腾芯片、维谛中标亿级项目、英伟达发布新GPU架构
- **技术突破类**：行业级技术突破
  - 例：钠电池成本跌破0.3元/Wh、液冷PUE突破1.05、单机柜功率突破200kW

#### ❌ 不触发推送（等周一再说）
- 常规中标项目（除非金额>10亿或客户是头部互联网）
- 厂家常规营销活动/展会
- 行业媒体常规分析文章
- 二线厂商普通产品发布
- 地方性政策（除非是北上广深等核心枢纽）

**推送格式**：
```markdown
🚨 【突发】数据中心行业动态

[摘要内容]

📎 [详细链接]
```
