# 数据中心行业动态 - 2026-05-28

> 本报告每周一更新，周二-周五仅跟踪重大突发新闻

## 🏢 数据中心行业动态

### 政策与趋势

• **PUE合规倒逼绿色转型**
  - 北京自2026年起对PUE>1.35的数据中心征收差别电价（超限一倍内加价0.2元/度，超一倍以上加价0.5元/度），推动存量数据中心改造
  - 📎 [北京市政府政策文件](https://www.beijing.gov.cn/gate/big5/www.beijing.gov.cn/zhengce/zcjd/202411/t20241118_3942744.html)

• **"算电协同"首次写入政府工作报告**
  - 数据中心与电力系统协同调度成为国家战略方向，AI算力电力需求预计到2030年翻倍至945TWh
  - 📎 [国际能源署报告 - 能源与人工智能](https://www.iea.org/reports/electricity-information-2024)

• **四部门联合推动液冷强制化**
  - 新建大型AI数据中心100%采用液冷散热，2028年前存量风冷全部改造，液冷渗透率成算力基地审批硬指标
  - 📎 [钛媒体 - 数据中心液冷正成为必选项](https://www.tmtpost.com/7989840.html)

### 核心厂商动态

#### 算力厂商

• **华为昇腾路线明确**
  - 公布未来三年芯片规划：2026年Q1推出昇腾950PR（自研HBM），2026年Q4推出950DT，2027年Q4推出960，2028年Q4推出970
  - 📎 [证券时报](https://stcn.com/article/detail/3345926.html) | [21经济网](https://www.21jingji.com/article/20250919/herald/333b8009e4388a029b22ad54e7104dee.html)

• **英伟达全栈化转型**
  - 从GPU供应商向全栈式AI工厂平台转型，与英特尔达成50亿美元合作，定制x86 CPU集成至AI基础设施
  - 📎 [21经济网](https://www.21jingji.com/article/20250919/herald/333b8009e4388a029b22ad54e7104dee.html)

• **CPU与GPU比例重构**
  - Intel与AMD明确指出，2026年数据中心CPU与GPU使用比例将从传统1:4向1:1甚至4:1演进
  - 📎 [新浪GPU热点](https://cj.sina.cn/articles/view/7857201856/1d45362c001905vv04?froms=ggmp)

#### UPS厂商

• **维谛（Vertiv）海外需求旺盛**
  - AI电力设施及液冷设备需求保持强劲，2026年全球数据中心供配电市场空间达730亿元
  - 📎 [新浪财经](https://stock.finance.sina.com.cn/stock/view/paper.php?symbol=sh000001&reportid=799973141724) | [维谛官网](https://www.vertiv.cn/zh-CN)

• **华为持续中标**
  - 在运营商集采中保持领先地位，依托模块化UPS、锂电融合、AI预测性维护等技术优势
  - 📎 [C114通信网](https://m.c114.com.cn/w118-1172423.html)

#### 制冷厂商

• **液冷产业化进入深水区**
  - 2026年液冷渗透率将突破37%，向2027年50%临界点迈进。冷板式液冷占65%（成熟度高），浸没式占34%（散热效率更高）
  - 📎 [钛媒体](https://www.tmtpost.com/7989840.html)

• **功率密度驱动液冷刚需**
  - 单芯片功耗从H100的700W跃升至Rubin的2000W乃至后续平台的5000W+，风冷在物理层面已触及天花板
  - 📎 [钛媒体](https://www.tmtpost.com/7989840.html)

• **新华三发布全域液冷整机**
  - S90000整机最高576核心，通过创新高密供电与全液冷散热范式，可将PUE降至1.05以下
  - 📎 [钛媒体](https://www.tmtpost.com/7989840.html)

#### 电池厂商

• **钠电池迎来规模化拐点**
  - 宁德时代、国轩高科等头部企业预计2026年Q4实现钠电池规模化量产，成本有望追平磷酸铁锂（当前钠电成本0.35-0.40元/Wh，铁锂0.30-0.35元/Wh）
  - 📎 [财联社](https://www.cls.cn/detail/2379770)

• **"锂钠协同"成为主流方案**
  - 海辰储能发布AIDC全时长储能解决方案，锂电负责8小时长时供电，钠电负责1小时内瞬时峰值调节，系统备电成本降低20%以上
  - 📎 [世纪新能源网](https://www.ne21.com/news/show-226960.html)

• **宁德时代签下最大钠电订单**
  - 与海博思创达成3年60GWh钠离子电池战略合作，目标直指储能领域
  - 📎 [世纪新能源网](https://www.ne21.com/news/show-226960.html)

• **中科院钠电安全突破**
  - 胡勇胜团队全球首次实现安时级钠离子电池"无热失控"，开发可聚合不燃电解质（PNE），150℃以上自动固化形成致密屏障
  - 📎 [低碳网](https://www.ditan.com/industry/energy-storage/101016.html)

### 技术前沿

• **液冷从"可选项"变"必选项"**
  - 当单机柜功率密度突破60kW甚至向130kW迈进时，风冷系统的散热极限与PUE劣化已不可逆转
  - 📎 [钛媒体](https://www.tmtpost.com/7989840.html)

• **AIDC场景催生钠电新机会**
  - AI训练的毫秒级功率波动让锂电池响应捉襟见肘，而钠电的高倍率放电、快响应、高安全特性完美适配AIDC短时备电需求
  - 📎 [世纪新能源网](https://www.ne21.com/news/show-226960.html)

• **浸没式液冷PUE可至1.1以下**
  - 中国电信试点显示，浸没式可使核心温度较风冷降低35%，PUE最低可达1.1
  - 📎 [钛媒体](https://www.tmtpost.com/7989840.html)

• **数据中心从"能耗大户"向"能源产销者"转型**
  - 通过采用液冷技术、绿电直供、余热回收等创新方案
  - 📎 [新浪财经](https://finance.sina.cn/2026-01-14/detail-inhhfmkx5309529.d.html)
