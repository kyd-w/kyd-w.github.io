# 中石油东方物探公司业务分析

## 公司概况
- **全称**：中国石油集团东方地球物理勘探有限责任公司（BGP）
- **位置**：涿州东方地球物理科技园
- **业务**：地球物理方法勘探油气资源为核心
- **规模**：营业收入连续两年保持全球物探行业第一，业务遍及全球59个国家

---

## 核心业务场景

### 1. 地震勘探数据处理（给地球做CT）
- **技术**：从二维地震→三维地震→宽方位高密度三维地震勘探
- **软件**：GeoEast系统（中国首个完全自主知识产权的超大型地震数据处理解释一体化软件）
- **数据量**：单体原始数据达PB级，一次计算任务数据量达十几个PB
- **计算特点**：
  - 计算量大且周期长（数周不中断计算）
  - 大量串行和并行任务
  - IO操作频繁
  - 7大软件包、18大技术系列、400多个功能模块
  - 源代码超过3000万行

### 2. 算力基础设施
- **涿州高性能计算中心**：
  - 物探行业亚洲最大的计算中心和数据存储中心
  - 2482节点共享资源池
  - **16.1 Pflops算力**
  - **90PB存力**
  - 全球算力资源"一张网"运行模式
- **处理解释互联网工厂**：
  - 2025年3月进入试生产阶段
  - 跨域远程生产（数据从千里外传到涿州算力云）
  - 解决跨域项目算力不足、无法快速扩充资源问题

### 3. AI转型（2025年重点方向）
- **与华为深度合作**：
  - 2023年10月启动GeoEast鲲鹏平台适配
  - 40天完成2000万行代码梳理编译
  - 鲲鹏较原处理器处理效率提升2-3倍
  - 基于昇腾加速卡的全波形反演（FWI）：
    - 一张昇腾卡半小时完成二维全波形反演（传统方法需6小时）
    - 二维/三维声波方程正演计算效率提升数倍
  - 计划2025年底前完成百亿参数三维模型首轮预训练
- **AI for Science**：
  - 数据与机理双驱动的波动方程智能求解
  - 基于昇思MindSpore AI4S使能套件
  - 自动微分技术 + 扩散模型的三维地质数据生成

### 4. 存储需求
- **浪潮AS13000存储系统**：
  - 34个节点
  - 整体容量达到4PB
  - 单节点读带宽1200MB/s以上
  - 写带宽1500MB/s以上
  - 分布式架构，支持未来轻松扩容

## 核心诉求预估

### 1. 算力升级与能效优化
- **痛点**：
  - 高密度采集使数据量暴增（PB级）
  - 处理周期长（数周），需要缩短
  - 功耗密度高，散热压力大
- **需求**：
  - 更高算力密度（单机柜功率持续提升）
  - 更低PUE（液冷可能是刚需）
  - 算力资源精准调配和算法优化

### 2. 国产化替代（信创）
- **现状**：
  - 已与华为深度合作（鲲鹏+昇腾适配）
  - GeoEast完成2000万行代码鲲鹏平台移植
- **需求**：
  - 继续推进国产算力平台替代
  - 供配电系统的国产化率要求
  - 液冷等新技术的国产化适配

### 3. 供配电可靠性
- **痛点**：
  - 地震数据处理不能中断（数周连续计算）
  - 一次故障损失巨大
- **需求**：
  - 高可靠性供配电方案（2N？DR？）
  - UPS/HVDC的可靠性保证
  - 快速响应的运维服务

### 4. 液冷散热
- **驱动因素**：
  - 单机柜功率持续提升（可能50kW-100kW+）
  - 传统风冷已到瓶颈
  - PUE要求更严格
- **需求**：
  - 浸没式液冷？冷板式液冷？
  - 制冷+供电一体化方案
  - 与华为液冷的对比选择

### 5. 数字化转型加速
- **目标**：
  - "处理解释互联网工厂"模式推广
  - 全球算力"一张网"优化
  - AI+物探技术工业化应用
- **需求**：
  - 数据中心智能化运维
  - 远程监控与管理
  - 能效优化与成本控制
