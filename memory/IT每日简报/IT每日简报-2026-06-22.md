## 3. Agent热门Skill

### 多智能体研究系统最佳实践

- **来源**: Anthropic官方技术博客
- **日期**: 2026-06-16
- **核心原则**:
  1. 根据查询复杂度动态分配资源（简单检索1个智能体，复杂研究10+个子智能体）
  2. 让智能体自我改进（Claude 4模型本身就是出色的提示词工程师）
  3. 并行工具调用极大提升速度（研究时间最多缩短90%）
  4. Token使用量是性能差异的关键因素（80%的性能差异由token使用量解释）

- **Notion**: 用户在工作区直接把编码、做PPT、整理表格这些活扔给Claude，几十个任务并行跑

- **Rakuten**: 一周部署一个专项智能体
- **Atlassian**: 接进Jira，开发者可以直接把任务分配给Claude智能体
- **Asana**: 做了AI Teammates，在项目管理里加入AI协作者
